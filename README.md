## 八、极客联盟核心目标

**技术路线
Geeks Union Technical Route**

**一、通过在gamefi赛道的持续深耕与沉淀，实现《赛车达人》在web3领域的进化与拓展，成为押注类经典模型的标准插件。**

通过在gamefi赛道的持续深耕与沉淀，实现《赛车达人》在web3领域的进化与拓展，成为押注类经典模型的标准插件。
游戏世界，经济模型中最基础的算法，包括生产力产出，集权统筹分配，和对赌押注与竞价拍卖。传统游戏中的逻辑算法通常被借用与链游之中，但都是各自算法，同时并不能实现区块链底层逻辑上的互通互认共享共用，这不符合区块链精神。对赌押注与竞价拍卖是一个常规算法，但却蕴含了Web3理念最基本的逻辑与经济分配。区跨链游戏是未来游戏的必然之路，对赌押注等基础算法也将成为链游的基础插件与标准协议。但目前业界并未完全关注这个方向，都在造应用，完全忽略了核心目标与核心意义以及核心价值所在。极客联盟的震撼发布将是Gamefi赛道链接基础设施与应用的一座里程碑式桥梁。

**二、构建基于《赛车达人》标准插件在gamefi领域的基本应用：实现第三方dapp嵌套与经济模型的适配。**

在链游初期，轻量级应用将会蓬勃发展，各类算法层出不穷，样式混乱，鱼龙混杂，迭代创新也罢，借壳乔装也罢，根本逻辑不会变。极客联盟专注于基本算法的通用型实践与开放，促成标准接口的落地与应用，实现在技术层与应用层的双向通用。通过通用性实践，极客联盟将可以在任意应用或DAPP中进行热插拔接入或嵌套使用。这将为链游进入爆发式增长提供些许动力源泉。同时，借助web3经济模型的理念，极客联盟将秉持开放式经济逻辑，用户可在本地自定义经济模型，这将促使任何使用极客联盟标准协议的用户以此搭建属于自己的模块化应用。
另外基于标准协议，极客联盟将对协议本身进行开放性策展，即用户在使用极客联盟标准协议后，将被默认为策展人。

**三、通过与各大游戏厂商碰触，实现在传统游戏渠道的可拓展，使得基于web3经济流转的标准插件具备自定义编程条件，以此为内置化SDK模块升级准备。**

标准插件中，协议是基础，但在大型游戏中，标准算法显然已不能满足多模态的场景需求，传统厂商的解决方案也在模块化发展，比如物理引擎，建模，渲染层等，也有趋向于本地化部署的策略方案。这些无疑是推动游戏产业发展变革的驱动。但在web3领域，这些东西都是中心化构架，没有一个可信可靠的东西将之联动，尽管游戏公链中可以解决一些经济模型的记账等，但并不能解决游戏内部的经济分配或有效标记与可信。这使得链游本身在底层是一个伪概念。
极客联盟标准插件与协议的诞生可以解决这个问题，但应用于大型场景中功能属性还是难以满足用户需求，为避免多次造车与可能对基础协议的反复调用或二次编译，极客联盟在满足自定义变成条件下，将对产品升级，实现SDK的开发，以为更多游戏厂商提供可热插拔的SDK标准模块。这大大降低了游戏开发中经济模型的二次编译等工作。用户只需要对标准模块进行设置即可满足经济游戏自定义经济模型。对于web3来说，这是一项划时代的创设。极客联盟将致力于此，力耕未来。

**四、《赛车达人》在完成SDK之前会通过cosmos进行一个过渡测试，并锻造一条可自由插拔的游戏基础设施，实现无缝跨链。打造链上链下的无门槛游戏体验。通过SDK可插拔的本地化部署方案，打造通过《赛车达人》游戏生态私有链，实现B端用户无托管式缔造自定义智能合约（游戏类协议）。实现策展协议的内置与经济模型分发。**

极客联盟在完成SDK之前会通过cosmos进行一个过渡测试，这是跨链的基础。同时也是标准插件与协议走向传统市场的一个重要过渡。
常规链游在实现链上行为时总会进行一个取舍，要么选择全上链，舍弃即时性，要么选择后者，只对有效资产上链。后者无疑对游戏体验是友好的，但不能做到经济互信，可以被视为完全中心化。
极客联盟便提供了这个问题的解决方案，基于高效Tps的加持及本地化部署，极客联盟为游戏端提供了开放的可编程选择，让经济模型在图灵黑箱中完备运营，消除了做恶可能。
同时极客联盟的轻量级专有链路，保证了网络响应的速率，让链游体验与传统游戏体验完全一致。
通过专有链路的加持与合约自定义部署，B的端用户可以无托管式实现经济模型的使用或流转。为创新型协议提供新的策展平台与经济场景。
通过以上技术实践，极客联盟将独立打造一套被动式策展协议，与极客联盟的有效交互行为将被标记度量，以实现上述经济场景的打造。这为创作者及用户提供了经济效益的最大化获益可能，值得期待。

**五、基于标准插件与协议的SDK化，与策展协议的有效应用，打造web3领域首家游戏产业基础插件的分发平台。**

游戏产业，无论手机端或者桌面端，游戏商城这种常见的分发平台之重要性无用多叙，这是游戏面向大众用户的直接窗口。极客联盟不是游戏商店，而且在web3世界为游戏提供基础设施的分发平台，如模拟引擎，图像渲染，素材元素，经济模型，与玩法规则，通用UI等，这些都是链游的基础设施，是通过极客联盟策展协议流转，同时拥有众多通过极客联盟SDK开发设计的经济模型和玩法规则。这些基础设施构建了整个Web3游戏帝国。
